# Sistema de Autogestión de Talento Humano

Proyecto académico desarrollado en el marco del programa
Tecnología en Análisis y Desarrollo de Software – SENA.

## Descripción
Sistema web orientado a pequeñas empresas para la gestión básica
de empleados, asistencia y solicitudes de vacaciones.

## Tecnologías
- HTML
- CSS
- JavaScript
- Backend y base de datos (en definición)
